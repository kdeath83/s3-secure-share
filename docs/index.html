How It Works

📤

Drag & Drop Upload

Drop files onto the web UI. Browser uploads directly to S3 via pre-signed URL — no file ever touches your server.

⏱️

Configurable TTL

Set link expiration per file: 1 hour, 24 hours, 7 days, 30 days, or never. Links automatically expire.

🔒

Password Protection

Optional per-file passwords. Hashed with PBKDF2-HMAC-SHA256. Verified with constant-time comparison.

🛡️

Rate Limiting

Per-IP rate limits prevent abuse: 10 uploads/min, 30 downloads/min. Built with slowapi.

📋

Security Headers

CSP, X-Frame-Options, HSTS, X-Content-Type-Options, Referrer-Policy on every response.

🔍

Audit Logging

Every action logged with file ID and IP address. Upload, download, delete — all tracked.

Direct upload

Security guarantee: The FastAPI backend never sees file contents. It only generates time-limited, cryptographically signed URLs that allow the browser to talk directly to S3. Files are stored in a private bucket with no public access.
